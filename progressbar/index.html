A pure CSS progress bar.

The purpose of this project is to demonstrate the capabilities CSS3 in modern web browsers, and as such this progress bar is styled entirely using CSS.

For funtional purposes the progression is handled by Javascript, allowing this to be used within your application right away.

No styles were harmed by images in the making of this project.
